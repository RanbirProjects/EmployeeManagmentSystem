## 📌 Project Overview

A **Full Stack Java Application** designed to manage employee records efficiently using modern backend architecture.

This project demonstrates:

- RESTful API Development  
- Clean Layered Architecture  
- Database Integration with JPA  
- Scalable Backend Design  

---

## 🏗️ Tech Stack

### 🔹 Backend
- Java  
- Spring Boot  
- Spring Data JPA  
- Hibernate  

### 🔹 Frontend
- HTML  
- CSS  
- JavaScript  

### 🔹 Database
- MySQL  

### 🔹 Build Tool
- Maven
